# 2048 Game

A smooth, animated implementation of the classic 2048 game using Python and Pygame.

## Features
- Smooth sliding animations
- Modern UI design
- Score tracking
- Game over detection

## Controls
- Use arrow keys to move tiles
- Press ESC to quit

## Installation
1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the game:
```bash
python game.py
```
